# 🚀 Front-End Developer Portfolio

A modern, fully responsive personal portfolio website built with React and Tailwind CSS. Showcase your projects, skills, and expertise with an elegant, performance-optimized design.

## 🌐 Live Demo

👉 **[View Portfolio](https://abdallahdev.vercel.app/)**

---

## 📌 Overview

This is a comprehensive, production-ready portfolio template built for front-end developers and web designers. It includes multiple sections to present your work effectively:

- **Home Section** - Eye-catching hero with introduction
- **About Section** - Tell your professional story
- **Skills Section** - Display your technical expertise with progress bars
- **Projects Section** - Showcase your best work with descriptions and links
- **Contact Section** - Easy ways for visitors to reach you
- **Responsive Navigation** - Works seamlessly on all devices

---

## ✨ Features

✅ **Fully Responsive Design** - Works perfectly on mobile, tablet, and desktop  
✅ **Dark Mode Support** - Professional dark theme toggle

✅ **Performance Optimized** - Built with Vite for fast load times

✅ **Mobile Menu** - Touch-friendly navigation for mobile devices

✅ **Easy to Customize** - Well-structured, commented code

## 🛠 Technologies & Dependencies

### Core Technologies

- **React** (v19.2.0) - UI library
- **Tailwind CSS** (v4.1.18) - Utility-first CSS framework
- **Vite** (v7.2.4) - Next-generation build tool
- **JavaScript ES6+** - Modern JavaScript features

### Additional Libraries

- **react-icons** (v5.5.0) - Icon library for UI elements
- **gh-pages** (v6.3.0) - Deployment to GitHub Pages

### Development Tools

- **ESLint** - Code quality and linting
- **Prettier** - Code formatting
- **React Plugin for Vite** - Fast HMR (Hot Module Replacement)
